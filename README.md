## Cara Menjalankan Notebook

### Via Google Colab (Dianjurkan)
1. Klik link notebook pada tabel di atas
2. Di halaman GitHub, klik tombol **"Open in Colab"** *(atau copy URL lalu buka di [colab.research.google.com](https://colab.research.google.com))*
3. Klik **Runtime → Run All** untuk menjalankan seluruh sel

### Via Lokal (Jupyter Notebook)
```bash
# Clone repository ini
git clone https://github.com/[username]/[nama-repo].git

# Masuk ke folder
cd [nama-repo]

# Install dependencies
pip install pandas numpy matplotlib seaborn scikit-learn scipy

# Jalankan Jupyter
jupyter notebook
```
